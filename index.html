Empang Indo
// Auto Farm Ikan · Kolam Management
STABLE

  • Beli ikan, cek stok seed sebelum beli (hanya beli kekurangan), tebar ke kolam, equip ember, beli & tebar pelet, tunggu panen, harvest semua, jual otomatis. Siklus berulang terus selama ada ikan di-ON.

  • Cek backpack dulu sebelum beli. Seed Arwana sudah ada 80, set maks 35 → langsung tebar tanpa beli. Seed cuma 20 → beli 15 saja biar genap 35. Hemat koin!

  • Ikan baru yang di-ON saat siklus berjalan masuk antrian dan diproses di siklus berikutnya. Perubahan maks ikan tidak mempengaruhi siklus yang sedang jalan.

  • Tiap ikan punya input angka langsung. Ketik berapa ekor sesuai upgrade kolam — default 20, maksimal 999. Berlaku di siklus berikutnya.

  • Setelah harvest selesai, langsung jual semua ikan otomatis. Tersedia juga tombol Jual Semua & Jual per-jenis di tab Fish untuk trigger manual kapanpun.

  • Nyala otomatis saat ada ikan yang di-ON, mati saat semua OFF. Jump tiap 4 menit, VirtualInput key tiap 60 detik, bypass event Idled Roblox.

  • HoldDuration semua ProximityPrompt di-set ke 0 — tidak perlu tahan lagi. Auto patch realtime untuk setiap prompt baru yang muncul di workspace.

  • Popup slide-in + suara ping saat ikan 100% siap panen. Tidak spam — satu kali per siklus per kolam, reset otomatis saat kolam kosong kembali.

  • Drag touch-safe, tombol lebih besar di mobile, tanpa UIStroke berat, panel compact. Minimize, resize, tombol Close. Single instance — execute ulang hapus instance lama otomatis.

Sawah Indo
// Auto Farm Tanaman · Multi Lahan
STABLE

  • Auto farm Padi, Jagung, Tomat, Terong, Stroberi. Beli benih otomatis, teleport ke posisi tanam, tanam burst, tunggu tumbuh, harvest semua sekaligus, ulangi terus.

  • Auto farm khusus Bibit Sawit & Durian di lahan terpisah (PlantLahanCrop remote). Berjalan paralel dengan Farm Auto 1 tanpa konflik, max 5 tanaman per jenis.

  • Tanam 5 bibit sekaligus per burst dengan delay 0.02s antar tanam dan 0.15s antar burst. Jauh lebih cepat dari metode satu per satu biasa.

  • Visual area lingkaran neon di dunia menandai titik tanam. Show Harvest Time tampilkan billboard di atas tiap tanaman: persentase tumbuh dan countdown waktu panen real-time.

  • Set Plant Pos [1] untuk farm reguler, Set Plant Pos [2] untuk lahan Sawit/Durian. Mode Saved (pakai posisi tersimpan) atau Current (ikut posisi karakter real-time).

  • Tombol Sell All menjual semua item hasil panen sekaligus via RequestSell remote. Satu klik bersih semua inventory panen.

  • Anti-AFK via VirtualUser bypass event Idled. Instant Interact via ProximityPromptService — trigger prompt langsung saat hold dimulai tanpa menunggu selesai.

Sambung Kata
// Auto Bot · Compe Mode · Select Word · Auto Join
STABLE

  • Bot otomatis cari kata terbaik berdasarkan ranking skor dari wordlist 50.000+ kata. Delay ketik bisa dikustom (min/max ms) agar terlihat natural. Retry otomatis hingga 6x jika kata gagal atau ditolak.

  • Mode kompetitif: bot sengaja pilih kata yang berakhiran sulit seperti "eo", "ea", "lt", "mp", "rp" dll agar lawan kehabisan opsi. Bisa pilih beberapa trap ending sekaligus dari dropdown multi-select.

  • Bot cek index koleksi kata kamu di server. Aktifkan Push Index → bot HANYA cari kata yang belum ada di koleksimu, sehingga setiap giliran menambah koleksi baru. Kata lama jadi fallback.

  • Mode pilih kata manual: muncul GUI draggable berisi daftar kata yang cocok saat giliran kamu. Klik kata → langsung diketik dan dikirim otomatis. Kata baru (belum di index) tampil duluan dengan warna hijau.

  • Bot sesekali salah ketik huruf tetangga di keyboard (20% chance per kata) lalu backspace dan koreksi. Membuat bot terlihat seperti manusia yang mengetik sungguhan, bukan bot sempurna.

  • Otomatis scan dan masuk meja yang tersedia sesuai mode yang dipilih: 2P, 4P, atau 8P — bisa multi-pilih. Bot prioritaskan meja yang hampir penuh. Jika lawan keluar, bot ikut leave dan cari meja baru.

  • Setelah match selesai, bot otomatis kembali join ke meja yang sama. Nama meja di-snapshot saat match mulai, dipakai ulang dengan retry loop hingga 10 detik jika meja belum siap.

  • Tambah kata tertentu ke daftar blacklist permanen — bot tidak akan pernah menggunakan kata tersebut. Ada juga blacklist otomatis sementara per match untuk kata yang ditolak server atau menyebabkan hit.

  • 5 metode sekaligus: VirtualUser, VirtualInputManager (tekan spasi), gerak karakter mikro, mouse1click, dan keypress fallback. Aktif saat event Idled + periodic tiap 10 menit sebagai backup.

  • Semua setting (delay, mode, blacklist, dll) bisa disimpan ke file JSON di executor dan dimuat kembali otomatis saat script dijalankan. Jika executor tidak support file I/O, config di-export ke clipboard.

  • Teleport satu klik ke Papan Hadiah (Papan13). Keybind buka/tutup UI bisa diubah sendiri. UI menggunakan WindUI dengan tag versi, toggle notifikasi, dan tombol Discord/WA di tab About.

Lengkapi Kata
// Auto Ketik · Instant Answer · Server Hop
STABLE

  • Bot otomatis mendeteksi soal yang muncul, mencari jawaban dari database kata, lalu mengetik dan mengirim jawaban secara otomatis. Delay ketik bisa dikustom agar terlihat natural dan tidak terlalu cepat.

  • Mode jawab instan: bot menjawab soal secepat mungkin begitu soal muncul tanpa delay tambahan. Cocok untuk grinding poin maksimal. Bisa dimatikan untuk mode natural yang lebih aman.

  • Auto deteksi server sepi (jumlah pemain di bawah threshold yang bisa dikustom). Jika server terlalu sepi, bot otomatis pindah ke server lain yang lebih ramai agar permainan terus berjalan lancar.
